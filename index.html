Each column = a sub-group. Width ∝ sample size. Hover for details.
Reading this chart: Each column represents a demographic sub-group. The column width is proportional to the number of survey respondents in that group. Within each column, colors from light to dark represent increasing numbers of sexual partners in the past year (0 → 11+). This Marimekko layout lets you compare both the composition (color distribution) and the prevalence (column width) simultaneously.
